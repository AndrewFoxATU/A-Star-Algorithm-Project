How it works: A* is a best-first search algorithm that finds the shortest path between two nodes. It uses a heuristic (Manhattan distance here) to guide the search, expanding the most promising node first. Nodes on the open list are candidates to explore; once a node is moved to the closed list it is finalized. The path is reconstructed by following parent pointers from the end node back to the start.
